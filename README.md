# Proj

## Descrição

Proj é uma visual novel desenvolvida utilizando Python através da engine Ren'Py.

O jogador assume o papel de um ex-funcionário que retorna a uma empresa abandonada após receber uma mensagem misteriosa. Durante a exploração do local, o jogador encontra sistemas ainda ativos, registros corrompidos e pistas sobre experimentos realizados pela empresa.

## Tecnologias Utilizadas

* Python
* Ren'Py

## Como Executar

1. Instale o Ren'Py.
2. Baixe este repositório.
3. Abra o projeto pelo Launcher do Ren'Py.
4. Execute o jogo.
